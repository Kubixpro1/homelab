# 🎟️ Seerr
Media request management for Jellyfin.

## 📦 Info
- Port: `5055`

## 🛠️ Setup Steps
1. Installed Seerr in the Jellyfin LXC
2. Connected to Jellyfin instance
3. Connected to Radarr.
4. Configured request permissions
