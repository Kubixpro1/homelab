# 🌊 qBittorrent
Torrent client used by Radarr for downloading.

## 📦 Info
- Port: `8080`

## 🛠️ Setup Steps
1. Installed qBittorrent in the arr-stack LXC
2. Configured download path
3. Connected to Radarr
